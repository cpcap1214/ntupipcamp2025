# NTU PIP Camp 2025 官方網站

這是國立臺灣大學 2025 年心理學系、資訊管理學系、物理治療學系聯合迎新宿營的官方網站。

## 功能特色

- 營隊資訊展示
- 團隊成員介紹
- 最新消息發布
- 文件下載中心
- 常見問題解答

## 技術架構

- HTML5
- CSS3
- JavaScript
- Font Awesome 圖標
- Google Fonts

## 本地開發

1. 克隆專案：
```bash
git clone https://github.com/YOUR_USERNAME/ntupipcamp2025.git
```

2. 進入專案目錄：
```bash
cd ntupipcamp2025
```

3. 使用瀏覽器開啟 `index.html` 檔案即可預覽網站。
